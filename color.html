HOW TO PLAY

SETUP
The grid starts empty. Each player clicks any empty cell to place their first orb (1 dot).

YOUR TURN
Click any empty cell to start there, or click one of your own cells to add a dot to it.

EXPLOSIONS
Each cell has a capacity equal to its number of neighbours (corners = 2, edges = 3, centre = 4). When a cell reaches capacity it explodes — it clears and fires one orb in each direction. Each neighbour gains +1 dot.

TAKEOVERS
If an exploding orb lands on an opponent's cell, that cell becomes yours and its count increases by 1. This can trigger chain reactions!

WIN
The game ends when only one player has cells remaining. That player wins!
